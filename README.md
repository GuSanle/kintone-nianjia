## 答题应用
- 获取space id
- 添加记录，修改记录时提交到统计应用
- 以space id为条件搜索答题统计应用是否已经存在记录
- 准备好以指定字母为开头的字段比如：'a_'，这些字段的值
- 已经存在记录则更新，不存在则添加
   
## 统计应用
- list页面 有个自动批改
- 点击批改，自动从答案的应用获取答案记录
- 获取所有的答题记录
- 循环每条记录，比对指定字母开头的字段的值
- 计算出每条记录的答题评分
- 生成批量更新记录，并更新

## 大屏可视化（备选）
- 每道题的正确率  柱状图
- 每个人的得分  折线图
- 最难的题目
- 最简单的题目
- 答题时间（最后更新时间）
